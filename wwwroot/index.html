SimpleHttpServer

IFTS 11 - desarrollo de software

Servidor HTTP implementado en .NET sobre Socket (capa de transporte), sin frameworks web adicionales. Parseo HTTP manual.

POST por formulario (Req. 6)

Completá el campo y presioná Enviar. El servidor loguea los datos y se muestra la confirmación sin salir de esta página.

Ejemplo: escribí Hola, esto es una prueba de POST en el campo.

GET con parámetros de consulta (Req. 7)

Los parámetros se envían en la URL y el servidor los registra en el log. Hacé clic en el enlace de abajo para probar:

/index.html?nombre=Anto

También podés escribir cualquier URL con ?clave=valor en la barra del navegador.

Compresión GZip (Req. 8)

El servidor comprime las respuestas si el navegador acepta gzip. Se verifica en el archivo LOG
