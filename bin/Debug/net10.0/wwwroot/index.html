SimpleHttpServer

IFTS 11 - desarrollo de software

Servidor HTTP minimalista implementado en .NET sobre Socket (capa de transporte), sin frameworks web adicionales. Parseo HTTP manual.

GET con parámetros de consulta (Req. 7)

Los parámetros se envían en la URL y el servidor los registra en el log. Hacé clic en el enlace de abajo para probar:

/index.html?nombre=Anto

También podés escribir cualquier URL con ?clave=valor en la barra del navegador.

POST por formulario (Req. 6)

Completá el campo y presioná Enviar. El servidor loguea los datos y responde con una página de confirmación mostrando lo recibido.

Ejemplo: escribí Hola, esto es una prueba de POST en el campo.

Compresión GZip (Req. 8)

El servidor comprime las respuestas si el navegador acepta gzip. Verificá el header Content-Encoding: gzip en F12 > Network, o revisá el log diario.
